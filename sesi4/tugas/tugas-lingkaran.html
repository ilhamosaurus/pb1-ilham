Cara Menghitung Luas dan Keliling Lingkaran

Rumus

r = jari-jari

Luas Lingkaran = phi x r x r

Keliling Lingkaran = phi x r x 2

Nilai Phi

Dalam tugas ini kita akan menggunakan nilai phi sebagai berikut: 3,141592653589793.

Langkah Mengerjakan

Luas Lingkaran

Luas Lingkaran = phi x r x r

Keliling Lingkaran

Keliling Lingkaran = phi x r x 2

nb: nilai hasil yang ditampilkan adalah nialah pembulatan dengan 2 angka dibelakang koma
